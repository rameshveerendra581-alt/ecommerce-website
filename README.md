Project Description

This project is a frontend-based E-Commerce Website developed as part of an internship project.
The website demonstrates the basic structure and features required for an online shopping platform, including product listing, cart, checkout, and contact functionality.

The project is built using HTML, CSS, JavaScript, and jQuery, following the given problem statement.
